Implémentations et code

Code minimal from scratch puis bibliothèques de production : scikit-learn, mlxtend, XGBoost et LightGBM.

1. Implémentation from scratch

La classe CustomStackingClassifier montre explicitement les OOF predictions. C’est la partie la plus importante à comprendre pour le devoir.

Ouvrir stacking_from_scratch.py Notebook from scratch

2. StackingClassifier — scikit-learn

stacking_lr = StackingClassifier(
    estimators=base_learners,
    final_estimator=LogisticRegression(max_iter=5000, random_state=SEED),
    stack_method="predict_proba",
    cv=5,
    n_jobs=-1
)

Cette implémentation est celle à utiliser en production académique lorsqu’on veut un code stable, intégré aux pipelines scikit-learn.

3. StackingCVClassifier — mlxtend

stacking_cv = StackingCVClassifier(
    classifiers=[model for _, model in base_learners],
    meta_classifier=LogisticRegression(max_iter=5000, random_state=SEED),
    use_probas=True,
    cv=5,
    random_state=SEED,
    n_jobs=-1
)
